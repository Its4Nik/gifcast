gifcast

Select an asciinema cast file, select a size, then click Render to generate an animated GIF.

The animated GIF can be saved by using your browser’s Save Image As... functionality, invoked by right-clicking the image.

Details

A larger selected size will result in a higher-resolution GIF.

Large images are scaled down to fit on the screen. Saving retains the full resolution.
